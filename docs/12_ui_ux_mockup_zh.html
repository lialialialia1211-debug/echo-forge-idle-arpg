你的回饋這版怎麼改
流派缺什麼不要做在表上，讓玩家自由發揮移除流派狀態面板。不做任何「還差幾件」的提示，流派完全交給玩家自己摸索。
戰鬥畫面只保留 HUD + 場地戰鬥畫面極簡化，只剩大圓場地 + 疊在角落的 HUD，其餘全部移出。
圓形場地要覆蓋螢幕至少 2/3場地放大置中，佔畫面主體；HUD 用「覆蓋」方式疊在角落，不佔版面、不擋戰鬥。
生命 + 魔力做成兩顆球疊在一起E（命）大球 + Flux（魔力）小球，小球覆蓋在大球上，合成一組雙球 HUD。
只有 BOSS 給畫面血條；菁英在小陀螺上給；其餘不給三級血條規則（見線框圖 A 與 §血條規則）。
刷圖跟 POE2 一樣，高難地圖需要 key新增「地圖 / 刷圖」畫面：POE2 式鑰匙裝置，高階地圖要插鑰匙，鑰匙有詞綴（更難更肥）。
養成畫面才好好調裝備/天賦/技能養成獨立成一個畫面，全屏處理裝備、天賦、技能。
新的三畫面結構：① 戰鬥（用看的，極簡）② 地圖/刷圖（POE2 鑰匙，決定去哪刷）③ 養成/工作台（全屏改裝備天賦技能）。戰鬥畫面只負責「呈現戰鬥」，決策都在另外兩個畫面。

虛線/實線 = 區塊示意（線框，非最終視覺）。畫面上除了場地與角落雙球 HUD，什麼都沒有 —— 沉浸在戰鬥本身。

雙球 HUD 的巧思：E（命，綠大球）與 Flux（魔，藍小球）疊成一組，小球覆蓋在大球右上。液面即時升降 → 一眼看能量。當 Flux 乾 → 死亡螺旋 觸發，兩球接觸處泛紅預警。放在左下角，完全不擋中央戰鬥。

血條顯示規則（三級）

敵人等級血條怎麼顯示為什麼
BOSS畫面頂部橫跨的大血條王戰是重點事件，值得一條專屬大血條營造壓迫感與里程碑感
菁英怪血條浮在該小陀螺正上方菁英值得盯，但不需搶走畫面；血條跟著它走，玩家自然聚焦
雜怪不顯示血條雜怪靠「停轉 = 死」表達即可；不給血條，畫面才乾淨、不淪為血條海

補充：我方的命/魔一律看左下雙球，不在陀螺上另掛血條，避免資訊重複。

選中一件 → 顯示前綴/後綴、與現有比較（↑↓）。不顯示「這推進了哪個流派」 —— 讓玩家自己判讀詞綴、自己組流派。

養成畫面才是密集資訊的家。這裡可以慢慢看、慢慢比、慢慢想 —— 因為不在 live 戰鬥旁邊，怎麼複雜都不影響戰鬥沉浸。

三畫面怎麼切

戰鬥地圖/刷圖養成

戰鬥打完 → 回地圖選下一張（要鑰匙就插鑰匙）→ 想變強就進養成 → 再回戰鬥。三個畫面各司其職，戰鬥畫面永遠保持乾淨。

設計定案（已拍板）

  • 雙球位置 → 左下角（定案）。
  • 鑰匙 → 可合成，但高難 BOSS 鑰匙只能靠掉落（定案）。一般鑰匙可用素材合成/升階；頂級王鑰維持稀有掉落，保住王戰的分量與稀缺性。
  • 雜怪死亡 → 直接碎裂消失（定案）。無血條、無倒地過場，致死即碎裂 fade out，畫面乾淨俐落。
  • 天賦節點 → 尚未設計。養成畫面的天賦頁先留空框，節點內容之後再設計。
  • 地圖呈現先用「格狀列表」；PoE 式節點圖集（Atlas）列為之後可選升級。
